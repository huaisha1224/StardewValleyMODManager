####zhihu_to_evernote

**将知乎收藏和"我关注的问题"自动发送到Evernote/印象笔记/OneNote/有道云笔记中**

###用到的库和程序
- Python_ver          = "Python3.6"
- selenium_ver        = "selenium3.141.0"
- Chrome_ver          = "Chrome73"
- chromedriver_Ver    = "2019/03/07版本"

###使用说明、
由于知乎的反爬虫机制、原来的requests模式已经不能抓取数据了、所以本次全部重新用selenium来抓取数据
